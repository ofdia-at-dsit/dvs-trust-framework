# UK digital verification services trust framework — version 1.0

This folder contains the **1.0 publication** of the UK digital verification services (DVS) trust framework, split into one Markdown file per main numbered section and grouped under the four published Parts.

The files in this folder mirror the official publication text as published by the Office for Digital Identities and Attributes (OfDIA). They are maintained here to make the framework easier to navigate, review, version and cite. The authoritative version is the publication on GOV.UK.
